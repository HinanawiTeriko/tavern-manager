## 一、项目定位

**地下城酒馆** — Godot 4.6 .NET **2D** 模拟经营游戏。

玩家是地牢小酒馆老板：白天去各地图采集材料、与NPC互动 → 夜晚在吧台后合成食物饮料，服务冒险者，赚取金币与声望。

- **引擎**: Godot 4.6.x .NET/Mono
- **渲染**: GL Compatibility (2D)
- **语言**: C# (.NET 8.0)
- **视角**: 2D 吧台视角（VA-11 Hall-A 风格：玩家在柜台后，客人在对面）
- **分辨率**: 1280×720，可缩放
- **设计系统**: "Hearth & Shadow" 暖色像素风配色（见 `scripts/ui/ThemeColors.cs`）

## 二、核心玩法循环

```
        白天（准备阶段）                    夜晚（营业阶段）
   ┌──────────────────────┐      ┌──────────────────────┐
   │ · 大地图选择目的地       │      │ · 客人陆续到来          │
   │ · 采集材料              │ ───→ │ · 拖拽材料 + 手势合成    │
   │ · 与 NPC 对话           │      │ · 上菜服务客人          │
   │ · 选择带上路的材料       │      │ · 获得金币/声望          │
   └──────────────────────┘      └──────────────────────┘
          ↑                              │
          └──────── 打烊结算 ←────────────┘
```

```
                    ┌─────────────┐
                    │  GameManager │  ← Autoload 顶层协调器

       ┌───────────────────┼───────────────────┐

   │ CraftSystem   │ │ GuestSystem │ │ EconomySystem   │
   │ (配方匹配)     │ │ (客人生成)   │ │ (金币/声望)     │
   └──────────────┘ └─────────────┘ └────────────────┘

```

### 各系统职责

| 系统 | 路径 | 状态 |

## 七、合成操作

### 拖拽流程
1. 鼠标左键从 ShortcutBar 拖材料到 CraftStation Slot1/Slot2
2. 右键退回单个材料 / 非拖拽时右键全部退回
3. 放置好后执行手势（加热/摇晃/搅拌）或点击"合成"

### 手势系统 (CraftStation)
| 按钮 | 操作 | 说明 |
|------|------|------|
| HeatBtn | 加热 | 按住 1.5 秒完成 |
| ShakeBtn | 摇晃 | 按住 1.5 秒完成 |
| StirBtn | 搅拌 | 按住 1.5 秒完成 |
| CraftBtn | 合成 | 无手势直接合成 |
| ServeBtn | 上菜 | 递交成品给客人 |
| ClearBtn | 清空 | 清空合成槽和成品槽 |

## 十、对话叙事

3 个 `.dialogue` 文件描述**莱恩**(见习骑士)的宿命故事线：

- `dialogue/d1.dialogue` — 教学日，莱恩首次登场
- `dialogue/d2.dialogue` — 宿命显现，账本显示莱恩明日会死；采集获得`迷睡花粉`；分支：干预/不干预
- `dialogue/d3.dialogue` — 因果结算，基于 d2 变量自动分叉

`NarrativeManager` 负责封装 Dialogue Manager 调用、管理变量、通过信号通知游戏逻辑。
