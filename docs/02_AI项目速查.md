## 一、项目定位

**地下城酒馆** — Godot 4.6 **2D** 模拟经营游戏。

玩家是地牢小酒馆老板：白天去各地图采集材料、与NPC互动 → 夜晚在吧台后合成食物饮料，服务冒险者，赚取金币与声望。

- **引擎**: Godot 4.6.x (标准版，非 .NET/Mono)
- **渲染**: GL Compatibility (2D)
- **语言**: GDScript
- **视角**: 2D 吧台视角（VA-11 Hall-A 风格：玩家在柜台后，客人在对面）
- **分辨率**: 1280×720，可缩放
- **设计系统**: "Hearth & Shadow" 暖色像素风配色（见 `scripts/ui/theme_colors.gd`）

## 二、核心玩法循环

```
        白天（准备阶段）                    夜晚（营业阶段）
   ┌──────────────────────┐      ┌──────────────────────┐
   │ · 大地图选择目的地       │      │ · 客人陆续到来          │
   │ · 采集材料              │ ───→ │ · 拖拽材料 + 手势合成    │
   │ · 与 NPC 对话           │      │ · 撒香料调味            │
   │ · 选择带上路的材料       │      │ · 上菜服务客人          │
   └──────────────────────┘      │ · 获得金币/声望          │
          ↑                      └──────────────────────┘
          └──────── 打烊结算 ←────────────┘
```

                    │  GameManager │  ← Autoload 顶层协调器

       ┌───────────────────┼───────────────────┐

## 七、合成操作

### 拖拽流程
1. 鼠标左键从 ShortcutBar 拖材料到 MixingArea
2. 右键退回单个材料 / 非拖拽时右键全部退回
3. 放置好后执行手势（加热/摇晃/搅拌）或点击操作按钮
4. 成品自动进入 ResultSlot
5. 可从 ResultSlot 拖到 SeasoningZone 撒香料，再拖给客人

### 手势系统 (CraftStation)
| 按钮 | 操作 | 说明 |
|------|------|------|
| 加热 | Heat | 按住 1.5 秒完成 |
| 搅拌 | Stir | 转圈 3 圈完成 |
| 摇晃 | Shake | 点击完成 |
| 倒出 | Pour | 点击完成 |
| 清空 | ClearBtn | 清空合成槽和成品槽 |

### 香料系统 (SeasoningZone)

三状态拖放台：

| 状态 | 显示 | 可拖入 |
|------|------|--------|
| 空 | 虚线框 + "放入成品" | 成品 |
| 有成品 | 成品名称 | 香料 |
| 已撒香料 | "成品名 · 香料名" | — |

## 十、对话叙事

重要 NPC 对话拆成 `.pre.dialogue`（点单）和 `.post.dialogue`（上菜反馈）两个文件。

`NarrativeManager` 负责封装 Dialogue Manager 调用、管理变量、通过信号通知游戏逻辑。
