# 酒馆统一内景空间设计

- 日期：2026-06-06
- 分支：`feat/blacksouls-intro`
- 状态：设计已确认，待生成室内空间母版
- 关联：`docs/superpowers/specs/2026-06-06-intro-art-pipeline-design.md`

## 目标

建立一套可同时服务开场动画和未来正式营业界面的酒馆内景。开场第二、三拍与营业吧台背景
使用同一建筑空间，但采用不同机位。任何视角都必须能通过入口门、左后酒架、右后壁炉、
L 形吧台和顶部木梁辨认出是同一间酒馆。

本设计不直接替换当前营业界面背景；先建立空间母版，开场美术批准后再为营业界面单独设计
符合客人立绘和物理工作面安全区的吧台视角。

## 核心视觉标志

石拱壁炉与炉火是酒馆的第一视觉标志，也是“熄灭与重燃”叙事的核心。其他家具和装饰必须
服从壁炉，不能形成第二个高亮中心。

空间辨识元素按优先级排列：

1. 右后方石拱壁炉。
2. 前景与右侧折入的 L 形厚木吧台。
3. 左后墙入口木门与低密度酒架。
4. 顶部 2–3 根粗木梁。
5. 少量悬挂干草药和克制的生活装饰。

## 固定空间布局

```text
后墙：
左侧入口门 -> 左后墙低密度酒架 -> 中央低对比人物区 -> 右后方石拱壁炉

顶部：
2–3 根粗木梁；只在左上或画面边缘悬挂最多两束干草药

中景：
简单桌椅沿两侧分布；中央保留清晰通道和人物舞台

前景：
L 形厚木吧台；长边服务营业视角，短边从右侧向壁炉方向折入
```

入口木门必须延续入口母版：

- 门位于空间左后方。
- 竖向木板，顶部和底部各一根横向木加固条。
- 中部靠自由边有一个横向黑铁拉手。
- 不出现招牌、墙牌、旗帜或文字。

## 家具与装饰预算

内景不能空，但装饰必须成组、低密度、可在 `320x180` 阅读：

- 酒架只有 2–3 层宽隔板。
- 每层只放少量大瓶罐剪影，不做满墙瓶海。
- 允许一个旧账本和一个小烛台。
- 干草药最多两束，只放在顶部边缘。
- 桌椅采用大轮廓，不画密集木纹。
- 室内禁止酒桶、板条箱、瓶堆、锅具墙、武器架和散落杂物。
- 中央人物区禁止高对比装饰和强光。

## 双机位接口

### 开场内景机位

镜头位于门槛内侧、略偏左，斜向观察室内。室内占画面至少 80%，入口门只在最左边保留
窄幅轮廓作为连续性锚点。L 形吧台的短边可见，并把视线引向右后方壁炉。

第二拍在此机位加入四名自然交流的顾客；第三拍严格复用同一机位和家具布局，删除人物并
熄灭主要暖光。

### 未来营业机位

玩家位于 L 形吧台长边后方，面向客人。固定运行时安全区：

- `x=440..840, y=80..440` 为主要客人立绘区，背景保持低对比。
- `y=410..655` 为物理吧台工作区，背景不能放关键叙事细节。
- 壁炉位于客人身后偏右，只提供轮廓光，不与立绘正面重叠。
- 入口门位于左后方。
- 酒架位于门与中央人物区之间，符合老板取酒动线。
- L 形吧台长边横贯下半屏，短边在右侧向壁炉方向折入。

营业机位不能直接复用开场静帧；它必须从室内空间母版重新构图。

## 开场顾客调度

第二拍使用恰好四名匿名顾客：

- 两人在一张桌旁以侧面或 3/4 剪影交谈。
- 一人侧坐，注意力落在酒杯或桌面。
- 一人靠近入口，呈现即将下井或刚刚归来的状态。

人物朝向必须不同。禁止所有人面向壁炉、对称围火、跪拜、祈祷或组成半圆。人物不画脸、
服饰细节、武器细节或个体肖像。

## 光照层级

- 壁炉是第一 amber 焦点，但在开场内景中位于右后方而非画面正中央。
- 小烛台和桌面酒杯只能提供极少量次级 amber 像素簇。
- 入口附近可以保留微弱冷暗轮廓。
- 至少约 70% 画面保持暗 teal 或煤黑。
- 禁止全画面橙色覆盖、柔光、体积光、景深和写实反射。

## 美术语言

- 以标题画面为严格风格参考。
- 使用低密度 chunky pixel clusters、大轮廓、阶梯边缘和硬阴影。
- 史诗感来自拱形壁炉、木梁尺度、留白和空间层级，不来自写实纹理。
- 石材、木材和瓶罐只保留少量成组高光，禁止均匀噪点和细碎材质。
- 任何候选图缩到 `320x180` 后，壁炉、吧台、门和酒架仍须一眼可辨。

## 生产顺序

1. 使用标题画面和入口母版生成无人物的 `tavern_interior_spatial_master`。
2. 审核建筑结构、L 形吧台、美感、装饰密度和原生尺寸可读性。
3. 从批准的室内母版编辑生成 `intro_hearth_memory`。
4. 从批准的暖景编辑生成完全同机位的 `intro_tavern_dark`。
5. 未来另立任务，从室内母版生成正式营业吧台背景。

## 完成标准

- 开场内景和未来营业视角可明确识别为同一空间。
- 壁炉、L 形吧台、入口门、酒架和木梁位置关系稳定。
- 室内有生活感但不拥挤，中央人物区保持低对比。
- 第二拍顾客自然交流，不呈现围火仪式。
- 第三拍能仅靠删除人物与暖光表达“灯灭了”。
- 设计在 `320x180` 原生网格上仍以大形状阅读。
