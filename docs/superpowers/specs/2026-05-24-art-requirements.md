# 地下城酒馆 — 美术需求文档

> 日期: 2026-05-24 | 版本: v1.0 | 关联策划: `docs/superpowers/specs/2026-05-24-tavern-2d-redesign.md`

---

## 1. 美术风格总纲

### 1.1 定位

**像素风 × 暗色调 × 地牢酒馆**。参考 VA-11 Hall-A 的吧台氛围，但更偏暗黑地牢质感——石壁、烛光、木纹、暖黄灯光与深紫阴影的对比。

### 1.2 核心关键词

`像素风` `地牢` `温暖酒馆` `烛光氛围` `中世纪奇幻` `暗色调` `手绘感像素`

### 1.3 技术规格

| 参数 | 值 |
|------|-----|
| 游戏分辨率 | 1280×720 |
| 美术风格 | 像素风 (Pixel Art) |
| 色深 | 低饱和 + 高对比，暗部偏紫/蓝，亮部偏暖黄 |
| 角色尺寸 | 立绘约 400×500px，实际像素画布建议 200×250 再 2x 放大 |
| 图标尺寸 | 材料/成品图标 64×64px (像素画布 32×32 再 2x) |
| UI 元素 | 64×64 至 256×128 不等 (像素画布 32×32 ~ 128×64 再 2x) |
| 背景尺寸 | 1280×720 (场景背景直接按此分辨率绘制) |

### 1.4 色彩策略

**主色调**：暖琥珀 (#CC8833 ~ #E8B84B) — 烛光、麦芽酒、木质吧台  
**暗部**：深褐紫 (#1A1020 ~ #2D1F2E) — 石壁、阴影、地牢氛围  
**点缀色**：
- 草药绿 #4A8C3F — 草药、藤蔓、部分 UI 强调
- 酒红 #8B1A2B — 葡萄酒、帘布、重要按钮
- 冷蓝 #3A5A8C — 月光、魔法物品、特殊 NPC
- 金色 #D4A843 — 金币、声望、高亮

**原则**：大面积低饱和 + 小面积高饱和点缀。背景不抢角色/UI 的视觉焦点。

## 2. 场景美术需求

游戏共 4 个场景，每个场景需求不同。

### 2.1 标题界面 (TitleScreen)

**当前状态**：有一张 AI 生成的占位背景图 (`dark_dungeon_tavern_interior_*.png`)，其余为默认 Godot UI。

**需求清单**：

| # | 资源 | 规格 | 说明 |
|---|------|------|------|
| T1 | **标题背景** | 1280×720 像素画 | 酒馆外景或吧台特写——厚重木门半开，暖黄光从门缝透出，门口有风吹动的招牌。整体暗调，光聚焦在画面中央。 |
| T2 | **标题字** | 512×128 像素字或位图字体 | "地下城酒馆" 四个字，风格：粗犷手写感，仿木质招牌刻字，暗金 + 边缘磨损。可做成带透明底的 PNG，叠加在背景上。 |
| T3 | **开始按钮** | 256×64 像素 | 木质按钮，hover 时微微发光/变亮 (两张图或 shader 调色)。文字"开始营业"直接画在按钮上。 |
| T4 | **装饰元素** | 64×64 各 2-3 个 | 烛台、悬挂的酒杯、冒险者徽章等小物件，散布在背景或 UI 层，丰富画面。 |

**场景文件**：`scenes/ui/TitleScreen.tscn`

## 5. UI 通用组件

### 5.1 按钮样式

统一使用像素风按钮，建议 3 种变体：

| 变体 | 用途 | 配色 |
|------|------|------|
| **主按钮** | 确认操作（出发、合成、上菜） | 暖褐底 + 金色边框 + 深色文字 |
| **次按钮** | 辅助操作（摇晃、加热、搅拌） | 暗褐底 + 石灰色边框 |
| **危险/取消** | 清空、退出 | 暗红底或暗灰底 |

所有按钮需提供 **普通态** 和 **按下态（或 Disabled 态）** 两张图。推荐尺寸：256×64 (宽按钮)、90×45 (小方形按钮)。

### 5.2 面板/弹窗

| 资源 | 规格 | 说明 |
|------|------|------|

| 面板标题栏 | 9-patch 可拉伸，高 32px | 面板顶部的深色标题条 |

### 5.3 字体

## 9. 文件命名与目录规范

所有美术源文件放入 `assets/` 目录，按以下结构：

```
assets/
├── textures/
│   ├── backgrounds/
│   │   ├── title_bg.png          # 标题背景
│   │   ├── tavern_bg.png         # 酒馆吧台背景
│   │   ├── daymap_bg.png         # 白天地图背景
│   │   └── ending_bg.png         # 结局背景
│   ├── characters/
│   │   ├── ryan/                  # 莱恩立绘 + 差分
│   │   ├── mira/                  # 米拉立绘 + 差分
│   │   └── guests/                # 普通客人头像
│   ├── items/
│   │   ├── materials/             # 材料图标 (ale, wine, bread, meat, herb)
│   │   ├── products/              # 成品图标 (按 key 命名)
│   │   └── props/                 # 剧情道具
│   └── effects/
│       ├── steam.png              # 蒸汽粒子
│       ├── splash.png             # 飞溅粒子
│       └── sparkle.png            # 完成闪光
├── ui/
│   ├── buttons/                   # 各种按钮
│   ├── panels/                    # 面板 9-patch
│   ├── bars/                      # 体力条、耐心条等
│   └── icons/                     # 通用小图标 (金币、声望等)
├── fonts/
│   └── pixel_zh.font              # 像素中文字体
└── audio/
    ├── bgm/                       # 背景音乐
    └── sfx/                       # 音效
```

**命名规范**：
- 全小写 + 下划线：`tavern_bg.png`, `ryan_smile.png`
- 图标按游戏内 key 命名：`materials/ale.png`, `products/meat_stew.png`
- 差分命名：`{角色}_{表情}.png`，如 `ryan_worried.png`

## 10. 补充说明

### 10.1 像素画制作建议

- **画布大小**：角色立绘建议 200×250 像素画布 + 2x 导出；图标 32×32 + 2x 导出；背景直接 1280×720 绘制。
- **调色板**：建议先确定一个 32-48 色的全局调色板，所有素材共用，保证风格统一。
- **轮廓**：角色和重要物件用深色（近黑）描边，背景物件不用描边以保持景深感。
- **光照**：所有素材统一假定光源来自上方偏前（吧台吊灯/烛光位置）。
