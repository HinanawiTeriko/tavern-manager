## 1. 美术风格总纲

### 1.1 定位

**像素风 × 暗色调 × 地牢酒馆**。参考 VA-11 Hall-A 的吧台氛围，但更偏暗黑地牢质感——石壁、烛光、木纹、暖黄灯光与深紫阴影的对比。

### 1.2 核心关键词

`像素风` `地牢` `温暖酒馆` `烛光氛围` `中世纪奇幻` `暗色调` `手绘感像素`

### 1.3 技术规格

| 参数 | 值 |
|------|-----|
| 游戏分辨率 | 1280×720 |
| 美术风格 | 像素风 (Pixel Art) |
| 色深 | 低饱和 + 高对比，暗部偏紫/蓝，亮部偏暖黄 |
| 角色尺寸 | 立绘约 400×500px，实际像素画布建议 200×250 再 2x 放大 |
| 图标尺寸 | 材料/成品图标 64×64px (像素画布 32×32 再 2x) |
| UI 元素 | 64×64 至 256×64 不等 |
| 背景尺寸 | 1280×720 (场景背景直接按此分辨率绘制) |

### 1.4 "Hearth & Shadow" 色彩策略

## 2. 已实现的设计系统 (`ThemeColors.cs`)

设计系统已通过 `scripts/ui/ThemeColors.cs` 实装到全部 4 个场景。后续新增 UI 直接调用即可，无需重复手写颜色。

### 2.1 按钮工厂方法

```csharp
ThemeColors.StyleButton(btn, fontSize);        // 一键套用琥珀按钮三态
ThemeColors.ButtonNormal(w, wBot);             // 单独获取 normal StyleBox
ThemeColors.ButtonHover(w, wBot);              // 单独获取 hover StyleBox
ThemeColors.ButtonPressed(wTop, w);            // 单独获取 pressed StyleBox
```

按钮三态设计：
- **Normal**: AmberPrimary 底色 + 下边框 4px 深色（模拟凸起）
- **Hover**: AmberBright 底色（变亮，模拟火光）
- **Pressed**: AmberDark 底色 + 上边框 4px 深色（模拟按下）

### 2.2 面板工厂方法

```csharp
ThemeColors.WoodPanel();        // 深木纹面板 (SurfaceMid 底色 + PanelBorder 边框)
ThemeColors.ParchmentPanel();   // 羊皮纸面板 (半透明暗底 + 琥珀弱边框)
```

### 2.3 文字工厂方法

```csharp
ThemeColors.StyleHeader(label, fontSize);   // 琥珀金标题 + 黑色 outline 描边
ThemeColors.StyleBody(label, fontSize);     // 浅色正文
ThemeColors.StyleDim(label, fontSize);      // 弱化副文字
```

### 2.4 使用位置

| 场景 | 文件 | 应用的样式 |
|------|------|-----------|
| 标题界面 | `TitleScreen.cs` | StyleHeader (标题), StyleButton (开始营业), TitlePanel (SurfaceMid 半透明) |
| 白天采集 | `DayMapView.cs` | StyleHeader (天标签), StyleButton (出发/继续/+/-), ParchmentPanel (结果面板) |
| 夜晚酒馆 | `TavernView.cs` | StyleButton (菜单/打烊/配方/背包), ParchmentPanel (菜单弹窗), 全套文字标签 |
| 结局结算 | `EndingScreen.cs` | StyleHeader (三十日酒馆记), StyleButton (退出/重开), 琥珀金/浅色文字 |

## 10. 设计系统接入指南

新增 UI 元素时，使用 `ThemeColors` 静态类（`scripts/ui/ThemeColors.cs`）：

```csharp
// 按钮
ThemeColors.StyleButton(myButton, fontSize: 16);

// 标题
ThemeColors.StyleHeader(titleLabel, fontSize: 28);

// 正文
ThemeColors.StyleBody(bodyLabel);

// 副文字
ThemeColors.StyleDim(hintLabel);

// 面板
myPanel.AddThemeStyleboxOverride("panel", ThemeColors.WoodPanel());
myPanel.AddThemeStyleboxOverride("panel", ThemeColors.ParchmentPanel());
```

所有颜色常量直接引用：
```csharp
label.AddThemeColorOverride("font_color", ThemeColors.TextLight);
label.AddThemeColorOverride("font_color", ThemeColors.AmberPrimary);
rect.Color = ThemeColors.BackgroundDeep;
```

## 11. 补充说明

### 11.1 像素画制作建议

- **画布**：角色立绘 200×250 + 2x 导出；图标 32×32 + 2x 导出；背景直接 1280×720
- **调色板**：使用 1.4 节色板，所有素材共用以保证统一
- **轮廓**：角色用深色描边，背景不用
- **光照**：统一假定光源来自上方偏前（吧台吊灯/烛光）
