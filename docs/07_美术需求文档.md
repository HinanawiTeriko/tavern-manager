- **游戏**: 2D 像素风模拟经营，地牢酒馆老板视角
- **分辨率**: 1280×720，所有背景直接按此分辨率绘制

- **像素画布**: 角色 200×250 → 2x 导出为 400×500；图标 32×32 → 2x 导出为 64×64
- **文件格式**: PNG-24，透明背景（角色/图标/UI），不透明（背景）
- **命名**: 全小写 + 下划线，如 `tavern_bg.png` / `ryan_smile.png`
- **交付**: 按 P0 → P1 → P2 分批，每批一个 zip

## 1. 美术风格总纲

### 1.1 核心关键词

`像素风` `地牢` `温暖酒馆` `烛光氛围` `中世纪奇幻` `暗色调` `手绘感像素`

### 1.2 通用规则（所有资产遵守）

| 规则 | 说明 |
|------|------|
| **光源** | 统一假定上方偏前（吧台吊灯/壁挂烛台），高光在物体顶部，阴影在底部 |
| **轮廓** | 角色/物品用 1-2px 深色描边（`#0d0b0a`），背景元素不用描边 |
| **饱和度** | 大面积低饱和（背景/面板），小面积高饱和（火焰/图标/按钮） |
| **对比度** | 前景（角色/UI）比背景高 20-30% 对比度，确保可读性 |
| **像素感** | 避免过度抗锯齿，保留像素边缘锐度；渐变用抖动（dithering）不用平滑渐变 |
| **色板统一** | 所有资产从本文档第 2 节色板取色，不引入新主色调 |

### 1.3 参考作品

| 作品 | 参考点 |
|------|--------|
| **VA-11 Hall-A** | 吧台构图——玩家在柜台后，客人在画面前景对面。角色立绘放在画面中央偏上。暗色背景 + 霓虹/暖光点缀 |
| **Darkest Dungeon** | 暗色调处理——深黑阴影、高对比光影、哥特式石墙纹理、角色粗黑描边风格 |
| **Coffee Talk** | 饮品制作 UI 布局、暖色照明下的吧台氛围 |
| **潜水员戴夫** | 白天/夜晚场景色调过渡（白天偏冷蓝灰 → 夜晚偏暖琥珀） |

### 2.1 主色板 "Hearth & Shadow"

```
  琥珀系（交互、火光、强调）          暗面系（背景、面板）

  │ #cc7300 暗琥珀 │ pressed/阴影    │ #231f1d 中   │ 标准面板
  │ #4b2800 深棕   │ 琥珀上文字      │ #2e2927 高   │ 进度条背景
  └──────────────┘                  │ #393431 最高 │ 悬浮面板
                                    └──────────────┘
  文字系                              功能色
  ┌──────────────┐                  ┌──────────────┐
  │ #eae1dd 正文 │ 主要文本          │ #4a8c3f 成功 │ 正确/完成

        帧1→4: 蒸汽从槽口冒出 → 上升 → 扩散 → 消散

```
文件:     vfx/splash_{帧号}.png

效果:     小液滴向左右飞溅，颜色用合成槽内的材料色（动态着色）
        帧1→3: 液滴出现 → 飞散 → 消失

文件:     vfx/swirl_{帧号}.png
