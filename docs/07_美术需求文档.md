- **游戏**: 2D 暗黑卡通模拟经营，地牢酒馆老板视角
- **分辨率**: 1280×720，所有背景直接按此分辨率绘制

- **图标规格**: 直接 64×64 px 手绘；小图标 24×24 px

- **命名**: 全小写 + 下划线，如 `tavern_bg.png` / `ryan_smile.png`
- **交付**: 按 P0 → P1 → P2 分批，每批一个 zip

## 1. 美术风格总纲

### 1.1 核心关键词

`手绘卡通` `哥特暗黑` `昏黄暖光` `烛光氛围` `中世纪奇幻` `赛璐珞质感` `夸张动效`

### 1.2 通用规则（所有资产遵守）

| 规则 | 说明 |
|------|------|
| **光源** | 统一假定上方偏前（吧台吊灯/壁挂烛台），高光在物体顶部，阴影在底部。全部暖光，不用冷色补光 |
| **轮廓** | 角色/物品用手绘线条描边，1-3px 粗细变化，带笔触感，颜色深暖褐 `#2d2018`。背景元素不用描边 |
| **饱和度** | 大面积低饱和（背景/面板），小面积高饱和（火焰/图标/按钮） |
| **对比度** | 前景（角色/UI）比背景高 20-30% 对比度，确保可读性 |
| **手绘感** | 线条有粗细变化和笔触感，填色干净（赛璐珞风格）。允许平滑渐变和柔光，但要保留手绘有机感 |
| **色板统一** | 所有资产从本文档第 3 节色板取色，不引入新主色调 |

### 1.3 参考作品

| 作品 | 参考点 |
|------|--------|
| **Don't Starve (饥荒)** | 暗黑卡通人物比例——瘦长剪影、粗犷笔触、夸张表情、深黑阴影处理 |
| **VA-11 Hall-A** | 吧台构图——玩家在柜台后，客人在画面前景对面。角色立绘放在画面中央偏上。暗色背景 + 暖光点缀 |
| **Cuphead** | 手绘赛璐珞风格的线条粗细变化和填色方式（仅技法参考，不参考角色设计） |
| **Coffee Talk** | 饮品制作 UI 布局、暖色照明下的吧台氛围 |

### 1.4 与 v2.0 (像素风) 的主要区别

| 维度 | v2.0 像素 | v3.0 暗黑卡通 |
|------|-----------|--------------|
| 线条 | 统一 1-2px 描边 | 手绘线条，1-3px 粗细变化 |
| 边缘 | 保留锯齿锐度 | 有机手绘轮廓 |
| 渐变 | 抖动 dithering | 允许平滑渐变 |
| 阴影色 | 偏冷紫/蓝 `#1a151c` | 暖褐 `#2d2018` |
| 缩放 | nearest-neighbor 2x | 直接目标尺寸绘制 |
| 动效 | 帧动画为主 | 帧动画 + UI 弹性交互 |
| 字体 | 像素点阵字 | 手写体标题 + 衬线/无衬线正文 |

## 2. 动效规范

### 2.1 按钮交互动效

按钮需要以下状态贴图支持交互动效：

| 状态 | 触发 | 动效描述 | 时长 |
|------|------|----------|------|
| **Normal** | 默认 | 按钮本色，无描边 | — |
| **Hover 进入** | 鼠标移入 / 选中 | 按钮轻微抖动一次 + 切换为 hover 底色 + 出现白色边框 | 抖动 0.15s |
| **Hover 离开** | 鼠标移出 | 按钮再抖动一次 + 回到 normal 底色，白色边框消失 | 抖动 0.15s |
| **Pressed** | 按下 | 颜色回到 normal 底色（下沉感） | 持续按住 |
| **Released** | 松开（仍在按钮上方） | 回到 hover 底色 + 白色边框 | 即时 |
| **Disabled** | 不可用 | 去饱和/变灰，无 hover 反馈 | — |

### 2.2 弹性动效（待细化）

以下交互动效方向已确定，具体弹性参数待后续迭代：

- 面板/弹窗出现时的入场过渡
- 合成成品出现时的视觉反馈
- 数值变化（金币/声望）的跳动过渡
- 角色表情切换的过渡方式
- 口味标签/状态提示的出现和消失

### 2.3 动效交付物

- 按钮 Hover 态贴图（白色边框版本）：`btn_wide_hover.png` / `btn_small_hover.png`
- 按钮 Pressed 态贴图（回到 normal 色）：`btn_wide_pressed.png` / `btn_small_pressed.png`
- 动效的具体实现参数由代码侧通过 Tween/AnimationPlayer 控制，美术侧交付状态贴图即可
