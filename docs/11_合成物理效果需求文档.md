## 概述

为当前物理吧台（BarWorkspace）的拖拽、投料、容器加工和后续香料撒入加入反馈，让操作有分量感和真实感。全部为视觉表现增强，不改变已有配方、容器和叙事结算逻辑。

---

## 物理效果清单

### 1. 抓起摇晃

**触发**：鼠标按下，从快捷栏或槽位拎起材料/物品时。

**效果**：
- 物品被拎起瞬间向两侧轻微摆动 1-2 次（幅度约 5-8px，频率约 0.1s）
- 像被绳子提起的重物，底部有重力拉拽的延迟
- 摆动衰减后稳定在鼠标下方

**参数建议**：

| 参数 | 值 |
|------|-----|
| 摆动幅度 | ± 8px |
| 摆动次数 | 1-2 次 |
| 衰减时长 | ~0.3s |

---

### 2. 拖拽惯性

**触发**：按住鼠标拖拽材料/物品移动时持续生效。

**效果**：
- 物品不 1:1 跟随鼠标，有轻微的位置延迟
- 鼠标快速移动时，物品落后约 10-15px，停下来时物品会慢慢追上
- 模拟实际重量感

**参数建议**：

| 参数 | 值 |
|------|-----|
| 跟随延迟（lerp 权重） | 0.15-0.2 |
| 最大落后距离 | 15px |

**实现思路**：物品实际位置每帧向鼠标位置做 lerp/平滑插值，而非直接赋值。

---

### 3. 松手下落

**触发**：将材料拖入合成区槽位并松开鼠标时。

**效果**：
- 物品从鼠标位置自由落体掉进槽位中心
- 落到槽位底部时出现一次弹跳（压缩 + 回弹）
- 弹跳衰减后停留在槽位中心

**参数建议**：

| 参数 | 值 |
|------|-----|
| 下落时长 | ~0.15s |
| 弹跳峰值高度 | 槽位 20% 高度（约 10-15px） |
| 弹跳次数 | 1 次 |
| 弹跳衰减 | ~0.2s |

**顺序**：下落 → 触底弹跳 → 稳定。

---

### 4. 脱靶回弹

**触发**：拖拽物品松手，但鼠标不在任何有效槽位范围内时。

**效果**：
- 物品自动飞回快捷栏原位
- 飞行过程带轻微弧线（抛物线），而不是直线归位
- 到达原位时有一次小弹跳

**参数建议**：

| 参数 | 值 |
|------|-----|
| 飞行时长 | ~0.25s |
| 飞行曲线 | 二次贝塞尔（弧线） |
| 落地弹跳 | 小幅度，1 次 |

---

### 5. 香料粉末

**触发**：在主合成区撒香料时（移动香料罐、进度条填充期间）。

**效果**：
- 香料罐下方持续飘落粉末粒子
- 粒子从罐口产生，向下飘落，颜色匹配香料颜色
- 粒子有小量水平随机偏移（模拟自然飘落）
- 粒子落地后消失

**参数建议**：

| 参数 | 值 |
|------|-----|
| 粒子生成频率 | 每 0.05-0.1s 一个 |
| 粒子大小 | 2-4px |
| 下落速度 | 30-50px/s |
| 水平漂移 | ± 10px 随机 |
| 粒子颜色 | 辣=红, 香草=绿, 盐=白, 花粉=紫 |
| 粒子生命周期 | ~0.5s |
