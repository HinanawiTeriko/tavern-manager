# 游戏总览

> 本文档回答"这是什么游戏"——供所有后续策划文档引用。

---

## 一句话描述

**地下城酒馆** 是一款以地牢世界为背景的 **3D 第一人称酒馆模拟经营游戏**。

---

## 概念定位

玩家扮演一间地牢深处小酒馆的老板。白天在酒馆内外自由活动——采集食材、与商人交易、和冒险者对话接取任务、装修扩建；夜晚开门营业，在吧台后根据客人订单合成食物与饮料，赚取金币与声望，逐步将一间破败酒馆经营成地下城最著名的冒险者据点。

**关键词**: 模拟经营 / 合成 / 叙事驱动 / 昼夜循环 / 地牢世界观

---

## 游戏体验目标

1. **吧台后的忙碌与掌控感** — 夜晚客人涌入，在一分钟内完成合成、递出成品，体验高峰期的紧张与满足
2. **经营的成长曲线** — 从只卖麦芽酒和面包的破店，到拥有烤炉灶台、几十种配方的知名酒馆
3. **地牢世界的沉浸感** — 客人都是冒险者，他们带来的故事串联成酒馆的日常与命运
4. **选择的重量** — 某些抉择（如莱恩的宿命线）没有完美答案，只有代价不同的后果

---

## 核心玩法循环

```
          白天阶段                          夜晚阶段
    ┌──────────────────┐           ┌──────────────────┐
    │ 自由移动/探索      │           │ 客人排队上门       │
    │ 采集食材           │           │ 查看订单需求       │
    │ 与商人买卖          │  ─────→  │ 拖拽材料合成       │
    │ 与 NPC 对话/接任务  │           │ 递交给客人         │
    │ 装修/扩建酒馆       │           │ 获得金币/声望       │
    └──────────────────┘           └──────────────────┘
           ↑                                  │
           └──────── 进入下一天 ←─────────────┘
```

## 技术规格

| 项目 | 规格 |
|------|------|
| 引擎 | Godot 4.6.x .NET/Mono |
| 渲染 | Forward+ (3D) |
| 语言 | C# (.NET 8.0) |
| 视角 | 第三人称自由移动 + 吧台固定视角交互 |
