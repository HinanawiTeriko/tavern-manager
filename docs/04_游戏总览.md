# 游戏总览

> 本文档回答"这是什么游戏"——供所有后续策划文档引用。

---

## 一句话描述

**地下城酒馆** 是一款以地牢世界为背景的 **2D 吧台模拟经营游戏**。

---

## 概念定位

玩家扮演一间地牢深处小酒馆的老板。白天去各地图采集食材、与冒险者对话推进剧情；夜晚在吧台后根据客人订单合成食物与饮料，赚取金币与声望，逐步将一间破败酒馆经营成地下城最著名的冒险者据点。

**视角**: VA-11 Hall-A 风格——玩家在吧台后，客人出现在柜台对面。

**关键词**: 模拟经营 / 合成 / 叙事驱动 / 昼夜循环 / 地牢世界观

---

## 游戏体验目标

1. **吧台后的忙碌与掌控感** — 夜晚客人涌入，在耐心耗尽前完成拖拽合成、手势操作、递交成品，体验高峰期的紧张与满足
2. **经营的成长曲线** — 从只卖麦芽酒和面包的破店，到掌握 9 种配方的知名酒馆
3. **地牢世界的沉浸感** — 客人都是冒险者，他们带来的故事串联成酒馆的日常与命运
4. **选择的重量** — 某些抉择（如莱恩的宿命线）没有完美答案，只有代价不同的后果

---

## 核心玩法循环

```
        白天阶段                          夜晚阶段
   ┌──────────────────┐           ┌──────────────────┐
   │ 选择目的地         │           │ 客人排队上门       │
   │ 采集食材           │           │ 查看订单需求       │
   │ 与 NPC 对话/接任务  │  ─────→  │ 拖拽材料合成       │
   │ 选择携带的材料      │           │ 手势操作(加热等)   │
   └──────────────────┘           │ 撒香料调味         │
          ↑                       │ 递交给客人         │
          │                       │ 获得金币/声望       │
          │                       └──────────────────┘
          │                                │
          └──────── 打烊结算 ←─────────────┘
```

## 技术规格

| 项目 | 规格 |
|------|------|
| 引擎 | Godot 4.6.x 标准版 |
| 渲染 | GL Compatibility (2D) |
| 语言 | GDScript |
| 视角 | 2D 固定吧台视角 |
| 分辨率 | 1280×720，可缩放视口 |
| 平台 | Windows / macOS / Linux / Web |
| 插件 | Dialogue Manager (对话)、Godot MCP (AI辅助) |
| 设计系统 | "Hearth & Shadow" 暖色像素风 |
