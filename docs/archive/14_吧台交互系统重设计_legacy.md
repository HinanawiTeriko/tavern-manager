## 1. 概述

将当前基于**固定槽位 + 操作按钮**的吧台合成交互，重设计为**自由放置桌面 + 物理工具**的统一交互模型。

三个核心目标：

- **统一工作面**：酿造、食物加工、调味全部在同一个吧台桌面上靠拖拽完成
- **物理工具替代按钮**：喷火枪、搅拌棒、酿造桶作为摆放在桌面上的可交互物体
- **服务叙事**：酿造调配系统通过"配方书填空"机制成为 NPC 故事线的载体

---

## 2. 核心变化

| 旧系统 | 新系统 |
|--------|--------|
| 固定 Slot 槽位（Slot1/Slot2/ResultSlot） | 吧台自由放置区，物品可随意摆放 |
| 操作按钮（加热/搅拌/摇晃） | 桌面工具物体（喷火枪/搅拌棒/酿造桶） |
| 独立 SeasoningZone 香料区 | 香料直接拖到吧台成品上撒 |
| 天数酿造（投料→等天数→取货） | 魔法速酿（投料→确认→摇晃→产出） |
| 配方列表已知全部 | 配方书填空式，靠 NPC 线索解锁 |

## 3. 吧台工作面

### 3.1 布局

```
┌──────────────────────────────────────────────────────────┐
│  TopPanel (金币/声望/天数/打烊)                           │
├──────────────────────────────────────────────────────────┤
│                                                          │
│                    客 人 区                               │
│              [立绘] [订单气泡] [耐心条]                    │
│                                                          │
├──────────────────────────────────────────────────────────┤
│  吧台工作面 (自由放置区)                                   │
│                                                          │
│  [喷火枪]   [搅拌棒]   [酿造桶]                           │
│                                                          │
│  [配方书]                                                │
│                                                          │
│  物品可自由拖放、临时堆叠在桌面任意位置                     │
├──────────────────────────────────────────────────────────┤
│  快 捷 栏 (0-9)  ← 材料/物品从这里拖出                    │
└──────────────────────────────────────────────────────────┘
```

### 3.2 基本规则

- **无固定槽位**：从快捷栏拖出的物品直接落在吧台任意空位，带有碰撞边界
- **自由排列**：多个材料/半成品/成品可同时堆在桌面上
- **退回操作**：右键点击吧台上物品 → 退回快捷栏
- **清理按钮**：桌面角落一个"清空桌面"按钮，一键回收所有物品

## 4. 工具系统

三个固定工具摆放在吧台桌面上，不可拖拽移动。

### 4.1 工具总览

| 工具 | 用途 | 用于 | 交互方式 |
|------|------|------|----------|
| **喷火枪** 🔥 | 加热 | 食物（烤肉、烤面包等） | 拖物品上去→进度条 1.5s |
| **搅拌棒** 🥄 | 搅拌混合 | 食物（汤、炖菜等） | 拖物品上去→搅拌动画 1s |
| **酿造桶** 🛢️ | 速酿+摇晃 | 饮品 | 逐个投材料→确认→摇晃出成品 |

### 4.2 喷火枪

**流程**：
1. 从桌面拖物品到喷火枪上
2. 出现加热进度条（~1.5s）
3. 需要保持鼠标按住，进度条持续填充
4. 中途拖走 → 取消，不消耗材料
5. 满格 → 完成，物品变化（如生肉→烤肉）

### 4.3 搅拌棒

**流程**：
1. 从桌面拖物品到搅拌棒上
2. 自动播放搅拌动画（~1s）
3. 完成，物品变化（如生肉+麦芽→肉汤）

### 4.4 多步骤加工

支持工具串联——一个加工完的产物可以继续被拖到另一个工具上：

- 生肉 → 喷火枪 → 烤肉 → 搅拌棒 → 混入汤中

## 5. 酿造桶

饮品生产专用工具，取代旧的天数酿造模型。

### 5.1 操作流程

```
快捷栏拖材料 → 逐个投入酿造桶 → 桶侧彩色刻痕 → 
材料匹配已知配方 → 桶身发光("可酿造") → 
玩家点击桶确认 → 桶进入"摇晃中" → 
玩家按住摇晃(左右快速拖拽) → 速酿特效 → 成品弹出桶旁
```

### 5.2 投料反馈

- 每投一个材料，桶侧面出现对应颜色的刻痕标记（颜色对应材料：麦芽金黄、葡萄紫红、草药翠绿等）
- 所有材料组合匹配某个已知配方时 → 桶身发光
- 材料组合不匹配任何配方 → 桶冒 `?` 气泡，不触发酿造
- 投放不相关材料的处理：同样冒 `?`，不消耗材料

### 5.3 摇晃操作

- 点击确认后桶进入可摇晃状态
- 玩家按住桶并左右快速拖拽鼠标来摇晃
- 摇晃速度越快，速酿完成越快
- 成品从桶口弹出，落在桶旁边的吧台上

### 5.4 速酿特效方向

桶身微微震动 → 桶口冒泡 → 闪光 → 成品弹出。约 1-1.5s。

## 6. 原料加工（食物线）

沿用现有合成规则，交互方式改为工具拖拽。

| 产品 | 材料 | 工具 |
|------|------|------|
| 烤肉 | 生肉 | 喷火枪 |
| 面包 | 面粉 | 喷火枪（烤制） |
| 肉汤 | 生肉 + 麦芽 | 搅拌棒 |
| 草药茶 | 草药 | 搅拌棒 |
| 肉夹面包 | 烤肉 + 面包 | 搅拌棒（混合） |

其他现有配方映射到新工具后原则上保持一致，不做加减。

## 7. 调味

- 香料从快捷栏拖出
- 直接拖到吧台上的成品即可撒入
- 香料附加对应的食物属性（辛辣/清香/咸香/安眠），覆盖式叠加
- 属性随物品传递，用于叙事判断

## 8. 配方书与叙事集成

### 8.1 配方书结构

三种状态的配方：

| 状态 | 显示 | 说明 |
|------|------|------|
| **已解锁** | 名称 + 完整材料列表 | 游戏开始时默认解锁的基础配方 |
| **部分已知** | 名称已显示、材料有空白槽 `□` | NPC 对话中获取线索后出现 |
| **锁定** | 显示 `???` | 尚未触发相关线索 |

### 8.2 填空机制

- NPC 在对话中透露口味线索（如"喝过一种发光的紫色酒"、"喉咙像火烧"）
- 配方书不自动更新——玩家记住线索后，去配方书中手动尝试填空
- 从快捷栏拖材料到空白槽位 `□` 中填入
- 所有空白填对 → 配方解锁，配方完整显示
- 填错不提示、不消耗材料

### 8.3 叙事使用

- 已解锁的配方可在关键叙事节点被 NPC 点名
- 玩家去酿造桶按配方制作 → 上菜 → 触发分支剧情
- 调配成功与否可导向不同角色关系和结局走向
- 与现有"物品属性判定命运"机制互补——调配是"关系铺垫"，物品属性是"结果判定"

## 9. 交互总结

### 9.1 全部操作一览

| 操作 | 方式 |
|------|------|
| 取材料 | 从快捷栏拖出到桌面 |
| 加热 | 把物品拖到喷火枪上→按住等进度条 |
| 搅拌 | 把物品拖到搅拌棒上→自动搅拌 |
| 投料入桶 | 把材料逐个拖到酿造桶 |
| 确认酿造 | 点击发光的酿造桶 |
| 摇晃出酒 | 按住桶左右拖拽 |
| 撒香料 | 把香料直接拖到成品上 |
| 上菜 | 把成品拖到客人区 |
| 退回物品 | 右键点吧台物品 |
| 清空桌面 | 点清理按钮 |

### 9.2 物品状态

物品在吧台上有独立碰撞体积，可被后续物理效果增强（惯性拖拽、碰撞反弹等），但基础交互不依赖物理。
