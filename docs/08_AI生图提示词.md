# 地下城酒馆 — AI 生图提示词

> 基于 `docs/07_美术需求文档.md` v3.0 | 2026-05-27
> 
> **重要**：v3.0 美术风格已从像素风切换为暗黑卡通手绘。以下提示词仍为旧版像素风格式，使用时请将风格描述替换为：
> 
> - `pixel art` → `hand-drawn dark cartoon` / `cel-shaded`
> - `crisp pixel edges` / `no anti-aliasing` → `hand-drawn variable-width outlines, clean fills`
> - `nearest-neighbor 2x` → 直接目标尺寸绘制
> - `Darkest Dungeon meets VA-11 Hall-A` → `Don't Starve gothic cartoon meets cel-shaded animation`
> - 阴影色从偏紫 `#1a151c` → 暖褐 `#2d2018`
> 
> 每个提示词可直接复制到 Midjourney / DALL-E / Stable Diffusion 等工具使用。
