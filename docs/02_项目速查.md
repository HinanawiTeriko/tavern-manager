# 项目速查

## 场景

| 场景 | 路径 | 说明 |
|---|---|---|
| 标题 | `scenes/ui/TitleScreen.tscn` | 新游戏、继续、设置入口 |
| 白天地图 | `scenes/ui/DayMap.tscn` | 地点选择、采集、商店、传闻、调查入口 |
| 酒馆 | `scenes/ui/Tavern.tscn` | 菜单准备、物理吧台、客人服务 |
| 结算 | `scenes/ui/LedgerScreen.tscn` | 夜晚结果、命运提示、重写今日 |
| 结局 | `scenes/ui/EndingScreen.tscn` | 结局展示 |

## Autoload

| 名称 | 路径 | 责任 |
|---|---|---|
| `GameManager` | `scripts/game_manager.gd` | 系统装配、场景流、玩法入口、存档、重开 |
| `DialogueManager` | addon uid | Dialogue Manager 插件 |
| `TutorialManager` | `scripts/tutorial/tutorial_manager.gd` | 教程状态和教程触发 |

## 关键数据

| 数据 | 用途 |
|---|---|
| `data/items.json` | 物品、材料、成品、价格、profile |
| `data/recipes.json` | 容器配方 |
| `data/locations.json` | DayMap 地点、奖励、贴文、商店 gossip |
| `data/rumors.json` | 地点传闻、菜单提示、客群权重 |
| `data/guest_group_profiles.json` | 匿名客群、偏好、备用订单、立绘池 |
| `data/regular_customers.json` | 30 名常客、表情立绘、台词、特质 |
| `data/guest_appetites.json` | 常客胃口偏好 |
| `data/food_attributes.json` | 菜品标签 |
| `data/inference_puzzles.json` | 推理题、线索、结论 |
| `data/npcs.json` | 重要 NPC、记忆、结局文案 |

## 常用入口

| 目标 | 方法 |
|---|---|
| 开始白天 | `GameManager.start_day_map(day)` |
| 访问地点 | `GameManager.visit_day_location(location_id)` |
| 获取今日传闻 | `GameManager.get_today_rumors()` |
| 确认菜单 | `GameManager.on_menu_confirmed(menu_items)` |
| 上菜 | `GameManager.request_serve(item_key, craft_style_data, seasoning_attribute)` |
| 重写今日 | `GameManager.restart_current_day()` |
| 新游戏 | `GameManager.new_game()` |
| 继续游戏 | `GameManager.continue_game()` |
