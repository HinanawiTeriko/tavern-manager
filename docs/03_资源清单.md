## 二、设计系统：Hearth & Shadow

### 工厂方法

| 方法 | 返回 | 用途 |
|------|------|------|
| `ButtonNormal()` | StyleBoxFlat | 按钮默认态 |
| `ButtonHover()` | StyleBoxFlat | 按钮悬停态 |
| `ButtonPressed()` | StyleBoxFlat | 按钮按下态 |
| `StyleButton(btn, fontSize)` | void | 一键应用三态 + 字体颜色 |

| `StyleBody(label, fontSize)` | void | 正文样式 |
| `StyleDim(label, fontSize)` | void | 次要文本样式 |

## 三、场景资源

### 3.1 TitleScreen（标题界面）

| 文件 | 路径 |
|------|------|
| 场景 | `scenes/ui/TitleScreen.tscn` |
| 脚本 | `scripts/ui/TitleScreen.cs` |

**节点结构：**
```
TitleScreen (Node2D)
├── Background (Sprite2D)          ← title_bg.png, 0.952×0.938 scale
├── DarkOverlay (ColorRect)       ← 35% 黑色叠加
└── UI (CanvasLayer)
    ├── TitlePanel (Panel)         ← 半透明 SurfaceMid 面板
    │   └── TitleLabel (Label)     ← "地下城酒馆" (Amber, 48px)
    ├── SubtitleLabel (Label)      ← "—— 冒险者的歇脚处 ——"
    ├── StartButton (Button)       ← "开始营业" (Amber 样式)
    ├── HintLabel (Label)          ← 提示文字
    └── VersionLabel (Label)       ← "v0.2 · 地下城酒馆"
```

### 3.2 DayMap（大地图）

| 文件 | 路径 |
|------|------|
| 场景 | `scenes/ui/DayMap.tscn` |
| 脚本 | `scripts/ui/DayMapView.cs` |


### 4.2 locations.json

定义大地图目的地（采集点和NPC对话）。字段：`Id`, `Name`, `Desc`, `Type` (Gather/Talk/Shop), `Materials`, `NPC`, `Day`.

### 4.3 npcs.json

定义 NPC 剧情配置。字段：`Id`, `Name`, `Scenes` (按 Day 分组的出现条件、对话文件、变量设置)。

### 4.4 inventory_default.json

定义初始库存（每种材料 999）。

## 七、输入映射

| 动作 | 按键 | 功能 |
|------|------|------|
| `menu_toggle` | E | 打开/关闭配方+背包菜单 |
| 拖拽 | 鼠标左键 | 从快捷栏拖材料到合成区 |
| 逐个退回 | 拖拽中鼠标右键 | 退回单个材料 |
| 全部退回 | 非拖拽时鼠标右键 | 退回合成区所有材料 |

## 九、构建与运行

```powershell
cd tavern-manager
dotnet build
# 用 Godot .NET 4.6.x 编辑器打开项目目录即可运行
```

- **.NET SDK**: 8.0+
- **Godot**: 4.6.x-stable .NET (Mono)
